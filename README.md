Welcome to Tap.me
==========

This is a group project for our final task at Makers. We are making a webapp that allows users to log in and either share their location or their feelings.

Tech stack
------

- Ruby on rails (backend)
- Javascript (frontend)
- Geolocation api
- Database (Postgres)
- Heroku
- Travis
- Capybara/minitest
- Trello
- css (bootstrap)
